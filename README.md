lua_run
=======

## android lua runer.

* 在安卓上编辑和运行lua

* 支持lua语法高亮

* 支持lua5.2.3版本

* 集成lua5.2离线文档

## 编译步骤


```
   $ cd lua_run
   $ android update project -p . -t android-19
   $ sh build_native.sh
```
